# 🚀 Next-Word Prediction: Energy-Aware Distributed Inference

## Overview
This internship project implements an intelligent next-word prediction system using advanced NLP models and edge-based distributed inference. The system improves typing efficiency by predicting the most probable next words while balancing prediction accuracy, latency, user experience, and energy consumption.

## Key Features
- Real-time next-word prediction
- Masked-word prediction
- Transformer-based NLP models
- Energy-aware distributed inference
- User-friendly web interface
- Edge-computing inspired architecture

2. Install dependencies:

3. Run:
   python app.py
4. Open:
   http://localhost:8000
